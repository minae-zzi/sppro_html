견적 요청, 패널 정보 검색, 자료실 활용 등의 기능을 제공하는 업무 플랫폼입니다.

주요화면 사용가이드

대시보드

좌측 메뉴: 기능별 빠른 접근
중앙 영역: 내 정보, 최근 게시글 현황, 실시간 패널 집계 현황
상단 바: 내 견적보기, 로그아웃 버튼

MS 정보검색

패널이 보유하고 있는 정보에 대한 조회
배너 위치, 조건설정 가능
사용은 엑셀의 피벗테이블과 동일

#1 Draft 견적

설문길이, 예상IR, 샘플수를 입력하면
예상되는 직간접비 확인 가능

#2 세부견적의뢰

1. 제목 입력
2. 의뢰하고자 하는 국내 정량 조사 또는 해외 정량 조사 선택
3. (해외 정량 조사의 경우) 원하는 해외 파트너사 선택
4. 국내, 해외 구분하여 요청할 조사 내용 입력
5. 해당 견적을 함께 보고싶은 수신자를 추가 가능
6. 첨부 파일 업로드 가능
7. [견적 의뢰하기] 클릭
8. 견적답변 완료 메일 수신하면 서치프로에서 견적 답변 보기

국내견적과 해외 파트너사별 견적을 분리하여 확인할 수 있어
더욱 편리합니다!

#3 진행상태 확인 방법

"내견적보기" 메뉴에서 요청 건 상태 확인 가능

답변대기

견적 회신 담당자가 아직 지정되지 않음

견적확인중

견적 회신 담당자가 지정되었고, 담당자가 견적 확인 중인 상태
국내·해외 견적 중 하나만 회신되도
상태는 [견적 확인 중]으로 유지되며,
실시간으로 해외 견적 회신 내용 확인 가능

답변완료

모든 견적의 답변이 완료됨

패널프로파일조사

패널 프로파일 조사를 진행하고 관리할 수 있는 기능입니다.
조사 설계, 실행, 결과 분석까지 전체 프로세스를 지원합니다.

MS패널자료실

마스터샘플 패널 관련 자료를 확인하고 다운로드할 수 있는 기능입니다.
패널 소개서, 통계 자료, 가이드 문서 등을 제공합니다.

KNOW-HOW

서치프로 사용법과 관련 노하우를 확인할 수 있는 기능입니다.
자주 묻는 질문, 사용 가이드, 팁과 노하우를 제공합니다.
